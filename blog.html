Question-3: What is event delegation in the DOM?

Event delegation is a technique where event listener add in parent element instead of every single child element . When we click child element it's bubbles up to parent and allowing event handling from the parent element

Question-4: What is event bubbling in the DOM?

Event bubbling is the concept of DOM when we receives an event that event bubbles up to its parent or any ancestor element until it reaches the root element.
